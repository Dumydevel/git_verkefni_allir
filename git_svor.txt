"#git spurningar og svör"

1. Hvað gera eftirfarandi Linux skipanir?
	cd= fer til diretory
	cd ~ = fer inn í heima svæði notanda
	ls=  sér öll diretory
	pwd= prentar vinnandi direotory
	mkdir= býr til direotory

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone= clónar geymslu í nýtt direotory
	git log= sýnir commit logs
	git status= sýnir vinnandi tré stöðuna
	git diff= sýnir breytingar milli vinnandi tré og commits
	git checkout= updatar brances 

3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff  											git diff sýnir breytingar milli vinnandi tré og commits
	b) git diff --staged 									git diff --staged sýnir líka breytingar
	c) git diff commit1 commit2								sýnir breytingar mill eina marga eða alla files
